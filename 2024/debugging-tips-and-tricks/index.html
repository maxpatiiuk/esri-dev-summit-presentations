ArcGIS Maps SDK for JavaScript:
Tips and Tricks for Developing and Debugging Apps

Presenters: Max Patiiuk, Noah Sager

Slides & Helpful Resources: https://bit.ly/esri-2024-debugging-tips-and-tricks

(Max) Welcome everyone to the session on ArcGIS Maps SDK for JavaScript!

My name is Max Patiiuk. I am a developer on the JavaScript team. I mostly work with Widgets and Web Components. And I am presenting with my colleague Noah Sager.

(Noah) Thanks Max. Hi everyone! My name is Noah Sager. I'm a Product Engineer on the JavaScript team. I work on several minor areas of the API, and a few major ones.

(Max) We have seen many developers create amazing applications with the power of the ArcGIS Maps SDK for JavaScript. The goal of this presentation is to give you great introductory resources and to set you on the path to creating impactful web mapping applications.

Today we will focus on the programming related aspects of creating applications with Maps SDK. We will cover many tips, tricks, and guides to debugging the apps you build.
